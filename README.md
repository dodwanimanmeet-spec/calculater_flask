
# 🧮 Calculator App

A simple calculator application built using Python.

## 📋 Description

This is a basic calculator app that performs arithmetic operations like addition, subtraction, multiplication, and division.

## 🚀 Features

- Addition ➕
- Subtraction ➖
- Multiplication ✖️
- Division ➗
- Simple and clean user interface

## 🛠️ Technologies Used

- Python 3.x
- Tkinter / Streamlit (GUI)

## ▶️ How to Run

1. Clone this repository:
   ```
   git clone https://github.com/your-username/calculator-app.git
   ```
2. Navigate to the project folder:
   ```
   cd calculator-app
   ```
3. Run the app:
   ```
   python calculator.py
   ```
